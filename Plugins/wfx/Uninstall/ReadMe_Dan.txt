UnInstaller v1.8.1 - FS-plugin for Total Commander 5.51 og nyere
----------------------------------------------------------------

Udvidet plugin for afinstallering af forskellig software. Det samme som
Kontrolpanel - "Tilføj\Fjern programmer", men med flere muligheder og mere bekvem.

Egenskaber:
------------
   - Vis alle registreringer om afinstallationsprogrammet
     (også skjulte) 
   - Afinstallation                           ("Enter")
   - Vis alle registreringer                  ("F3" eler "Ctrl+Q")
   - Slet ukorrekte links                     ("Del" eller "F8")
   - Rediger visse registreringer             ("Alt"+"Enter")
   - Konfigurer plugin                        ("Alt"+"Enter" i Netværkssteder)


Installation
------------
1. udpak arkivet til en tom mappe
2. Vælg Opsætnong - Indstillinger - Plugins - Filsystem-plugins - Konfigurer
3. Klik på "Tilføj"
4. Gå til den mappe, hvortil arkivet blev udpakket, og vælg UnInstTC.wfx
5. Klik OK. Du kan nu få adgang til pluginet i "Netværkssteder"
