-- Playlist plugin, version 1.06 (freeware)--

This lister plugin displays the contents of m3u and pls playlists with relatively detailed info about each song in the list (including id3 tag info for mp3 files). It can also play the files.

Use Ctrl + C to copy current song info to clipboard and Ctrl + A to copy list contents. These commands are located in both main and the context menu.

Press Enter to play the highlighted song (if the path is valid). Also: double-click. When it finishes playing the song, the plugin will go on to the next, but will not loop around the list.

Search function is present ( hit F7 or Ctrl + F for find, F5 or F3 for find next ).

Note: song durations are read from playlists, not the files themselves, so you will get "Unknown" for some of them. You can learn the song duration by playing it, but new information will not be written back into the list.
