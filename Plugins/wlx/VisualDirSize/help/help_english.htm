1) Description

  • VisualDirSize provides a graphical overview of the files/folders in a selected directory.
  • You can view a chart by putting the cursor on any directory and pressing <Ctrl+Q>.
  • The selected directory shall be scanned and the sizes of the files¦folders will be displayed as a bar chart in the Quick View panel.
  • Such a bar chart is useful to show the relative sizes of files¦folders.
  • For instance : you can quickly locate the biggest file¦folder in the directory.
  • Because the way in which the plugin works, it's not recommended to use it on very big directories.
  • The delays might be pretty significant.

2) Installation

  1. TC 5.51
Copy the <visualdirsize.wlx> file into your Total Commander plugins directory.
Then add the following line to [ListerPlugins] section in the wincmd.ini file:
0=C:\totalcmd\plugins\visualdirsize.wlx (Change the path to your real one )
  1. TC 6.5 and newer
This version of TC has an "auto install plugin" feature. Just strike Enter on the archive of the plugin and TC'll ask you whether you want to install it.
Using the free tool TC Plugman, you can also install the plugin very easily for all versions of TC quoted above.

3) Location of VisualDirSize.ini

  • <VisualDirSize.ini> shall be saved in the same directory as <Wincmd.ini> by default.
  • If you don't like this location you can simply move the file to one of the following destinations :
  • In a subdirectory called "VisualDirSize" relative to the directory where Wincmd.ini is located
  • The directory of the Plugin.
  • The directory of Total Commander.

4) Use

  • Put the cursor on any directory then press “Ctrl+Q”.
  • It's impossible to execute the plugin with F3 - it works ONLY with Ctrl+Q.
  • You can use the arrow-keysand Home, End, PgUp and PgDn or SPACE-BAR to scroll trough the bars.
  • You can do a mouse left double-click on a graph to land into the corresponding sub-folder and a right double-click to go up towards the parent folder.
  • To abort the scanning simply press longer the ESC key.

5) F. A. Q.

  • If after pressing Ctrl+Q there is still the Total Commander default list displayed for a directory-entry, showing the directory info as plain text, move VisualDirSize higher up in the list of the installed Lister plugins.

  • You have simply to wait until the plugin scans the selected directory and then select the next one.

6) Charts

  • VisualDirSize can visualize the folder sizes in four types of charts. A chart shows always the size of the sub-folders of the currently selected folder.
  • The charts can be sorted by size¦name and show the size¦percent¦number of files dependant on which kind of view-type the user selected in the tool-bar.
  • Double-clicking on a bar will land into the associated folder.
Bar chart Represent the results of scanning the currently selected drive¦folder graphically, with a bar-graph.
 
Pie chart Represents the results of scanning the currently selected drive¦folder graphically, with a pie (sectors) graph. In this mode small file¦folders may be summarized in a single slice named “Other”.
 
Block chart Represents the results of scanning the currently selected drive¦folder graphically, with colored blocks. If the block represents a directory, all files contained within that directory and sub directories are displayed as blocks representing them close it. All directories that are too small to be displayed individually get merged together.
 
Extensions chart Represents the results of scanning the currently selected drive¦folder as text with a bar-graph, generalized by file types (extensions).

7) Scan Filtering


  • The Scan Filtering feature allows you to filter the scanned files and folders.
  • With the scan filtering, you can specify precisely which files would be included in charts.
  • So, if you want to get information about certain file-types only, for example graphic files, use the filter "*.jpg *.gif" here.
File-name masks
  1. In this field, you can enter the search mask(s) for the files you want to find.
  2. Names with spaces must be put within double quotes, e.g. "Letter to Mr. Smith.doc", otherwise VisualDirSize should look for each part of the name, individually.
  3. You may use several filters separated by a semi-colon or a space.

Examples :

  • *.txt; *.exe (*.txt *.exe) finds all files with txt or exe extension
  • *.ini finds for example win.ini
  • Smith finds "Letter to Mr. Smith.doc"
  • The exclusion-masks are placed after a “|”.

Example :

  • w*.* | *.bak finds files which start with w and do not end with .bak
  • To Include¦Exclude certain directories into¦from the search, their names must have a trailing backslash \.

Example :

  • windows\ system32\ *.ini finds ini files only in the dirs. of windows\ and system32
  • *.ini | windows\ finds all ini files except those in directories named "Windows" and their sub-dirs.
TC Searches Use the TC Searches button at the right of the dialogue-box to see the parameters for frequent searches that Total Commander has saved.
 
Use Regular Expressions Search in file names using the “Regular Expressions”.  
Exclude matching files Exclude (rather than include) matching files.

These options will not be saved when you exit VisualDirSize in order to avoid wrong results when using it the next time.

8) Report generating

  • VisualDirSize has powerful built-in report exporting capabilities.
  • Exported report contents can often be imported or otherwise consumed by third-party applications for an additional processing.
  • A wide range of export formats are available, including :
HTML report Allows you to save a report as HTML file, which can be viewed with any HTML browser later.
 
TXT Save the collected data to a “Tab separated text file” that can be imported by applications like Microsoft Excel.
 
Only image The main application chart can be exported directly as a JPEG image-file.

Report exporting functionality can be accessed by pressing the Generate report button.

9) Configuration Dialogue

External icon library This option allows you to choose an external library from witch the default icons for files and directories will be loaded (you can use also the %COMMANDER_PATH% variable in the path).
 
Background colour Select the background colour for the charts.
 
Font colour Use this option to select the text colour for the charts.
 
Change font Choose a font for the charts in VisualDirSize. The font can be bold or normal, italic is NOT supported.
 
Sort like WinXP Uses mixed lexical and numerical sorting, e.g. file10.txt comes AFTER file1.txt. This option works only in Windows XP, in other systems it's normal sorting (this is not quite true).
 
Import TC Settings Import the above settings from the current Total Commander instance.
 
ColourTheme_x You can select a colour setting for the charts from that list.
 
Max Hue, Min Hue, Saturation, Lightness Colour theme for the bars. All values are in HSL (Hue, Saturation, Lightness). Bars' colors will be generated from
(Max Hue, Saturation, Lightness) to (Min Hue, Saturation, Lightness) in HSL.
 
Save current schema Here you can save the current colours setting. It shall be added at the bottom of the ColourTheme_x list.
 
Dont show more bars than Sets how many bars are displayed (default = 500). Reason :
The plugin scans all files and folders in selected directory but only the first 500 are displayed, because with slower machines, a problem with refreshing appears. If your computer is powerful enough, you can increase that value.
 
Scale factor in Block mode This option allows you to specify how much directory information gets "lumped" together.
A scale factor of 1 will draw exactly to the boundaries of the VisualDirSize plugin window.
A scale factor of 2 will draw twice as much information as can be displayed in the plugin window.
 
Size display Determines how the file size is shown in the main file lists. You can choose between
bytes         The size is shown in bytes, as in previous versions of Total Commander
kbytes       Size is shown in kbytes. 1 kbyte = 1024 bytes
Mbytes      Size is shown in Mbytes. 1 kbyte = 1024*1024 bytes
Gbytes       Size is shown in Gbytes. 1 kbyte = 1024*1024*1024 bytes
bkM            depeding on the file size, it is displayed in bytes, kbytes, MBytes or GBytes
bkM2          same as above, but with one digit after the comma to get a more exact size (default)
 
Launch browser after HTML report is generated Automatically, opens the HTML exports in the default web browser after its generation is completed.
 
 
Language Allows you to choose a language from the list. If youd like to translate VisualDirSize into your language, please email me. If your language isn't included in the package, contact me by email.
