Script Content plugin 0.2 'Lev Freidin (c) 2005-2008

Plugin allows to write your own vbs and js scripts to get information from the file or for parsing it's name and for placing the result to the plugin's column. Can be useful for getting useful information from the file, for sorting files in a desired order, for complex file renaming with Multy Rename tool.
See ini file for the examples of usage
change value in [Script] Section to define current rules
Refresh current window after ini file editing.

version 0.2 - is a test version, for checking it's work with more than one column
js scripts can work wrong
the results are retreived to "content", "content1", "content2" ... "content9" variables

If you want to create additional columns of the plugin then create a copy of a plugin directory. Rename wdx file, do not rename ini file. Register new wdx as new plugin and add a new column.
I'll try to add more columns in a future version of the plugin

Script Content plugin 0.2 'Lev Freidin (c) 2005-2008
Позволяет самому писать vbs и js скрипты для получения и обработки информации из файла или для обработки его имени и помещения результата в колонку плагина. Может быть полезен для получения полезной информации из файла, для сортировки файлов в желаемом порядке, для сложных переименований файлов в Multy Rename tool
См. в ini файле примеры применения 
Измените значение в [Script] Section чтобы определить текущие правила
После редактирования ini файла обновите текущее окно Total Commander

версия 0.2 - тестовая версия, для проверки работы более чем с одной колонкой
js скрипты могут работать неверно
результаты работы выводятся в переменные "content", "content1", "content2" ... "content9"

Если вы хотите дополнительные колонки плагина, то скопируйте директорию плагина и переименуйте wdx файл, имя ini файла оставить без изменений, зарегистрируйте как новый плагин и добавьте колонки.
